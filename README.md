# Polla Mundialista

## Recomendaciones para la API
### instalar paquetes de la API (solo la primera vez)
cd api/ && npm install

### iniciar la base de datos (debe estar instalado mongodb previamente)
cd api/ && npm run db
### iniciar entorno de desarrollo de la API 
cd api/ && npm run dev
